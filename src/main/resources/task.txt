Предметная область
Предположим, есть эфемерный заказчик, ему нужно автоматизировать управление и учёт в ресторане. Он так формулирует свою проблему:
"Повара заказывают какие продукты нужно доставить сейчас, или в будущем.
Кладовщики принимают продукты на склад и разносят требуемые сейчас продукты до кухни.
Снабженцы по будущим требованиям покупают продукты и доставляют на склад.".
Технологии и требования
Требуется разработать Web приложение по предметной области на Java 8.
Приложение должно быть самодостаточным и запускаемым (например: java -jar restaurant.jar)
Серверный framework - Spring (Spring Boot).
База данных - любая реляционная свободно распространяемая (PostgreSQL). Транзакции обязательны.
Клиентский framework - любой.
Взаимодействие с браузером - только через REST и JSON. Генерация HTML или CSS на сервере не разрешается.
Сборка с помощью Maven.
